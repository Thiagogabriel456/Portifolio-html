MINHAS ESPECIALIDADES.

HTML

A Linguagem de Marcação de Hipertexto (HTML) é uma linguagem de computador que compõe a maior parte das páginas da internet e dos aplicativos online. Linguagem aprendida na Universidade de Vila Velha (UVV).

CSS

O CSS é chamado de linguagem Cascading Style Sheet e é usado para estilizar elementos escritos em uma linguagem de marcação como HTML, o CSS separa o conteúdo da decoração do site. Linguagem aprendida na Universidade de Vila Velha (UVV).

UI/UX

Na criação de interfaces projetadas para o usuário, utiliza-se diversos elementos de design para promover consistência, facilidade de navegação, adaptabilidade e acessibilidade, além de combinar tudo isso com aspectos de UX (experiência do usuário).

JavaScipt

é o nome de uma linguagem de programação leve e de uso geral. Tal linguagem é baseada e objetos e que faz parte das principais tecnologias da World Wide Web (rede mundial de computadores). Linguagem aprendida na Universidade Vilha Velha (UVV).
